Border

Margin (.m-* / .m*-*)

  • *-auto => size: auto
  • *-1 => size: 0.25 rem
  • *-2 => size: 0.75 rem
  • *-3 => size: 1.25 rem
  • *-4 => size: 1.75 rem
  • *-5 => size: 2.25 rem
  • .mx => margin kanan dan kiri (sumbu x)
  • .my => margin atas dan bawah (sumbu y)

Padding (.p-* / .p*-*)

  • *-auto => size: auto
  • *-1 => size: 0.25 rem
  • *-2 => size: 0.75 rem
  • *-3 => size: 1.25 rem
  • *-4 => size: 1.75 rem
  • *-5 => size: 2.25 rem
  • .px => padding kanan dan kiri (sumbu x)
  • .py => padding atas dan bawah (sumbu y)
